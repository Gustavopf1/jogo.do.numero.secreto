# 🎯 Jogo do Número Secreto

Projeto desenvolvido durante o curso de **Lógica de Programação** da [Alura](https://www.alura.com.br/), onde aprendi a integrar **JavaScript** com **HTML** para criar interatividade em páginas web.

## 🕹️ Sobre o projeto

O **Jogo do Número Secreto** é um jogo simples de adivinhação no navegador. O sistema sorteia um número aleatório entre 1 e 10, e o jogador tenta descobrir qual é esse número com base em dicas fornecidas a cada tentativa.
